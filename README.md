# Optiprim3

**Technology-Enabled Logistics. Built by an Operator.**

Optiprim3 is a tech-forward logistics company built and operated by an owner-operator 
who runs both the truck and the tech stack. AI-powered dispatch, cloud infrastructure, 
and cybersecurity — all under one authority. No middlemen. No platform dependency.

---

## Tech Stack

- [React](https://react.dev/)
- [Three.js](https://threejs.org/)
- [GSAP](https://gsap.com/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Vite](https://vitejs.dev/)
- [Zustand](https://zustand-demo.pmnd.rs/)

---

## Local Development
```bash
git clone https://github.com/TrapNerdMont3ga/Optiprim3-landing.git
cd Optiprim3-landing
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
